# Principes Directeurs FLOW

Les principes directeurs constituent les règles structurantes du programme FLOW.

Ils servent à orienter les choix d'urbanisme, d'architecture et d'organisation du programme. Ils sont volontairement stables dans le temps et indépendants des solutions techniques retenues.
