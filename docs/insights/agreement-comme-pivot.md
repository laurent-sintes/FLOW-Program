# Agreement comme Pivot

## Constat

Les différences entre Achat, B2B et B2C ne proviennent pas principalement de la demande.

Elles proviennent du cadre d'engagement applicable.

## Agreement

Agreement représente notamment :
- contrat client ;
- accord fournisseur ;
- assortiment ;
- prix ;
- allocations ;
- SLA ;
- conditions de livraison.

## Conséquence

La plateforme FLOW doit séparer :

Agreement → Demand → Decision → Execution

plutôt que d'encapsuler toutes les règles dans les demandes.
