## Constat

Les OMS modernes regroupent historiquement :
- visibilité stock ;
- ATP ;
- sourcing ;
- allocation ;
- orchestration ;
- transport ;
- retours.

Le terme OMS est devenu un mot-valise.

## Insight

FLOW ne cherche pas nécessairement un nouvel OMS.

FLOW cherche à expliciter les responsabilités historiquement agrégées dans les OMS.

## Du Order Management au Demand Management

L'objet central devient la demande :
- commande B2C ;
- commande B2B ;
- retour ;
- transfert ;
- réapprovisionnement ;
- commande d'achat.

## Conséquence

FLOW est davantage une plateforme de Demand Management qu'une plateforme d'Order Management.
