# Insights

Cette section conserve les découvertes, hypothèses, convictions et enseignements du programme FLOW.

## Premiers insights

- FLOW n'est probablement pas un OMS.
- Les demandes sont plus stables que les processus.
- Agreement est le pivot qui explique les différences entre Achat, B2B et B2C.
- Inventory Visibility est une Shared Business Capability.
- Decision Services est une Shared Business Capability.
- Une plateforme doit fédérer autant qu'elle mutualise.
