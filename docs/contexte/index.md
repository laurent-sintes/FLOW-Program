# Contexte

Cette section rassemble les éléments de contexte nécessaires pour comprendre l'origine du programme FLOW.

Elle ne décrit pas encore la cible. Elle documente le terrain de départ : les patrimoines applicatifs, les disparités BRD / GBM, les études réalisées et les constats qui ont conduit à reformuler le projet.
